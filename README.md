# Snake Water Gun Game

This is a simple Python project based on the classic Snake, Water and Gun game. I created this project while practicing Python basics and improving my programming logic.

The game lets you play against the computer. Every round, the computer randomly chooses Snake, Water or Gun, compares it with your choice, and displays the winner along with the score.

## Features

* Play against the computer
* Random computer choices
* Score tracking
* Easy-to-use command-line interface

## Technologies Used

* Python

## What I Learned

While building this project, I practiced:

* Conditional statements
* Loops
* Functions
* Random module
* Basic game logic

This project helped me strengthen my understanding of Python fundamentals through a fun and interactive program.
